# Overlay Platform - Frontend

A Next.js frontend application for the Overlay Platform that displays AI-powered document analysis results.

## Features

- **Document Upload**: Upload contracts and documents for AI analysis
- **Real-Time Analysis**: View comprehensive analysis results from 6 AI agents
- **Overall Score Display**: See the weighted average score (0-100) with visual indicators
- **Criterion Scores**: Detailed breakdown of 7 evaluation criteria with progress bars
- **Clarification Questions**: View high-priority questions generated by AI
- **Strengths & Weaknesses**: Comprehensive lists of document pros and cons
- **AI Recommendations**: Actionable steps to improve the document
- **Structure Validation**: Compliance check against document templates
- **Grammar Analysis**: Detailed grammar and writing quality feedback

## Tech Stack

- **Next.js 15**: React framework with App Router
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first styling
- **shadcn/ui**: High-quality React components
- **Lucide Icons**: Beautiful icon library

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the application.
